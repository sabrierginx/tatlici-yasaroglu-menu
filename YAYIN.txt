TATLICI YAŞAROĞLU — Menüyü internet adresiyle yayınlama
=======================================================

Bu klasör tek sayfalık bir sitedir (index.html). Aşağıdaki yöntemlerden biriyle ücretsiz https:// adresi alırsınız.

--- Seçenek A: Netlify Drop (en hızlı, hesap gerekir) ---
1. https://app.netlify.com/drop adresine gidin.
2. Bu klasörün tamamını (içinde index.html olan) tarayıcı penceresine sürükleyip bırakın.
3. Size https://rastgele-isim.netlify.app gibi bir adres verilir; İsterseniz sitede "Domain settings" ile ismi değiştirebilirsiniz.
4. QR kodu bu tam https adresine yönlendirin.

--- Seçenek B: GitHub Pages ---
1. github.com’da yeni bir boş depo oluşturun.
2. Bu klasörde zaten "git init" yapıldıysa:
   git remote add origin https://github.com/KULLANICI/DEPO.git
   git branch -M main
   git push -u origin main
3. Depo → Settings → Pages → Source: "Deploy from a branch", branch: main, folder: / (root) → Save.
4. Birkaç dakika sonra https://KULLANICI.github.io/DEPO/ adresinde açılır.

--- Seçenek C: Cloudflare Pages ---
1. https://pages.cloudflare.com — projeyi bağlayın veya klasörü yükleyin.
2. Build command boş, output directory: / veya .

--- Yayın sonrası (isteğe bağlı) ---
index.html içindeki <head> bölümünde SITE_URL yorum satırını bulun; tam adresinizi yazıp paylaşım (og:url) için kullanabilirsiniz.
